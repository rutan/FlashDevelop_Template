■ なにこれ？

■ 使う前に
いじらないといけないのは，
・application.xml
　自分のアプリの設定にする
・bat/setupSDK.bat
　SDKのパスを自分の環境に合わせる
・bat/SetupApplication.bat
　自分のアプリに合わせる

プロジェクト名「Test」，パッケージ名無しで作ったプロジェクトなので，
その辺りは使う前に合わせないと困ることに．
